# Deep Ensembles for Predictive Uncertainty Estimation
 
Reproduction of selected experiments from
*Simple and Scalable Predictive Uncertainty Estimation using Deep Ensembles*
(Lakshminarayanan, Pritzel, Blundell, NeurIPS 2017).
 
This work was carried out as part of the Applied Statistical Inference
(ASI) course at EURECOM.

## Setup
 
```bash
git clone https://github.com/RP-13-p/deep-ensembles-asi.git
cd deep-ensembles-asi
python -m venv .venv
source .venv/bin/activate    # on Windows: .venv\Scripts\activate
pip install -r requirements.txt
```
